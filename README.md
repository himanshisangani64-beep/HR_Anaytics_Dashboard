#  HR Analytics Dashboard

An interactive **HR Analytics Dashboard** built using **Microsoft Excel Online**. This dashboard provides insights into employee demographics, attrition, income, education, overtime, and age distribution using interactive charts and filters.

##  Dashboard Features

- Employee Count KPI
- Average Employee Age
- Average Monthly Income
- Attrition by Gender
- Employee Count by Age Group
- Monthly Income by Age Group & Overtime
- Attrition by Education
- Department Filter
- Job Role Filter
- Interactive Dashboard
